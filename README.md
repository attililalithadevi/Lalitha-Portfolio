# Lalitha Devi Attili - Web Developer Portfolio

**Lalitha Devi Attili - Web Developer Portfolio**  
As a BCA graduate (CGPA 9.06) from Aditya Degree College for Women, I present my portfolio showcasing web development, Android, and Python skills. Explore my resume, projects like MoodLens and VoiceVibe, and achievements including Oracle & Microsoft certifications, plus NCAT (Rank 539) & APICET (Rank 1385) ranks. Built with HTML, CSS, and JavaScript, this responsive site reflects my passion for innovative solutions. View the code, check my works, and connect via the contact section!  

## Overview
This portfolio includes the following sections:
- **About Me**: A brief introduction to my background and skills.
- **Resume**: Details of my education and professional experience.
- **Works**: A collection of my projects, categorized into Web Applications, Android Applications, and Other Applications.
- **Achievements**: A timeline of certifications, competition ranks, and awards.
- **Contact**: A form to get in touch with me, along with a map and social links.

## Features
- Responsive design for desktop and mobile views.
- Interactive navigation with a sidebar and filterable project gallery.
- Timeline-based resume and achievements sections.
- Integration with GitHub repositories for project links.

## Prerequisites
- A modern web browser (Chrome, Firefox, Edge, etc.).
- Internet connection for loading external fonts and icons.
